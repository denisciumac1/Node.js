Modulele reprezinta o parte componenta foarte importanta in develompemtul modern JavaScript.

| functii: declararea, argumentii | instalarea modulelor: npm install/yarn add | preprocesoare css: sass/scss

In baza fragmentului de cod din din index executati urmatoarele puncte:
Prin comanda npm init initiati un nou "package.json"
Instalati oricare din modulele sass, node-sass
Verificati pe scurt documentatia modulului ales
Conectati modulul si complecati codul functiei compileSCSS() astfel incat, odata rulata aplicatia voastra, aceasta sa compileze fisierul src/style.scss si sa plaseze rezultatul intr-un fisier dist/style.css
